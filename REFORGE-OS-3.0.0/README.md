# REFORGE OS 3.0.0

## Enterprise-Grade Device Analysis & Compliance Platform

REFORGE OS is a professional, enterprise-grade platform for device analysis, legal classification, and compliance management. Built with modern technologies and best practices.

## 🚀 Features

- **Device Analysis**: Comprehensive device profiling and analysis
- **Legal Classification**: Automated legal classification and routing
- **Compliance Management**: Compliance reporting and audit trails
- **Ownership Verification**: Secure ownership attestation
- **Audit Logging**: Complete audit trail for all operations
- **Enterprise Architecture**: Scalable, maintainable, and testable

## 🏗️ Architecture

### Technology Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Desktop**: Tauri 1.5+
- **Backend**: Rust (Tauri) + Python (FastAPI)
- **Styling**: Tailwind CSS
- **Testing**: Vitest + Playwright

### Enterprise Features

- ✅ TypeScript strict mode
- ✅ Dependency injection
- ✅ Centralized error handling
- ✅ Structured logging
- ✅ Configuration management
- ✅ CI/CD pipeline
- ✅ Security best practices
- ✅ Comprehensive testing

## 📋 Prerequisites

- Node.js 20+
- npm 10+
- Rust (stable)
- Python 3.14+ (for backend)

## 🛠️ Installation

```bash
# Clone the repository
git clone <repository-url>
cd REFORGE-OS-3.0.0

# Install dependencies
cd "apps/Reforge os"
npm install
```

## 🚀 Quick Start

### Development

```bash
# Start development server
npm run dev
```

### Production Build

```bash
# Build for production
npm run build:prod
```
