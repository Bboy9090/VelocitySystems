# Bobby's Workshop (Layer 1 — Public UI)

**Purpose:** Trust, Education, UX, Community, Repair Intelligence, Reports

---

## What This Layer Is

Bobby's Workshop is the public-facing user interface of REFORGE OS. It provides:

- Device analysis dashboards
- Legal classifications
- Guided repair intelligence (non-circumventing)
- Customer-facing transparency
- Certification & training
- Compliance reports

---

## Technology Stack

- **Frontend:** React + TypeScript
- **Desktop:** Tauri (Rust + Web)
- **Styling:** Tailwind CSS + Custom Bronx Night Theme
- **API Client:** HTTP-based (ForgeWorks Core + Legacy APIs)

---

## Design System

**Theme:** Bronx Night (Super Mario-inspired)
- Primary Accent: Neon Cyan `#2DD4FF`
- Secondary Accent: Neon Magenta `#FF3DBB`
- Background: Midnight `#0F1114`
- Text: Soft Ash `#E8EAED`

**Animations:** Safe, non-action (fade, slide, pulse)

---

## Key Features

### Device Analysis
- Read-only device state analysis
- Ownership confidence scoring
- Security state classification

### Compliance Reporting
- Complete compliance summaries
- PDF export with branding
- Audit trail verification

### Legal Classification
- Jurisdiction-aware status
- Risk level assessment
- Authority routing guidance

### Custodian Vault (Gated)
- Interpretive review mode
- Ownership confidence ≥ 85% required
- Analysis-only, no instructions

## Compliance Rules

✅ **Allowed:**

❌ **Never:**
