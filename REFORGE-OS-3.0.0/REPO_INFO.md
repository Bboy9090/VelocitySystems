# Repository Information

## Repository Name

**REFORGE OS** (also known as **Bobby's Workshop 3.0** or **ForgeWorks Platform**)

---

## Core Features

### 1. **Repair Shop Management Platform**
A comprehensive, compliance-first platform for professional repair shop operations.

### 2. **Device Diagnostics & Analysis**
- **Android** devices (phones, tablets)
- **iOS/iPadOS** devices (iPhone, iPad)
- **Windows** PCs and laptops
- **Linux** systems
- **Mac** computers (MacBook, iMac)
- Universal device detection via **BootForge USB**

### 3. **Case Management**
- Customer intake and case creation
- Device tracking and management
- Status tracking
- Notes and documentation

### 4. **Diagnostics Workflow**
- Authorized device diagnostics
- Policy gates for compliance
- Ownership verification
- Diagnostic report generation
- ADB diagnostics for Android

### 5. **Recovery & Solutions**
- OEM firmware lookup
- Recovery guidance generation
- **Custodial Closet** - Solutions database for all device types
- Boot recovery workflows for crashed/unbootable systems
- Problem-solution mapping

### 6. **Evidence & Documentation**
- Evidence bundle generation (Apple Support, Carrier, Generic)
- Compliance documentation
- Audit trail creation
- PDF report generation

### 7. **Audit & Compliance**
- Immutable, hash-chained audit logging
- Complete activity tracking
- Compliance reporting
- Policy gate enforcement

### 8. **Policy Gates**
- Ownership attestation
- Device authorization checks
- No-circumvention enforcement
- Destructive action confirmations

### 9. **Frontend (React/TypeScript)**
- Beautiful, modern UI with design system
- 16+ page components
- Device intake interface
- Diagnostics interface
- Recovery guidance interface
- Audit log viewer
- Evidence bundle generator
- Solutions browser (Custodial Closet)

### 10. **Backend (FastAPI/Python)**
- RESTful API
- Case management endpoints
- Device detection endpoints
- Diagnostics endpoints
- Recovery endpoints
- Solutions API
- Audit log endpoints
- Evidence bundle endpoints

### 11. **Solutions Database**
- Problem-solution mapping
- Device type categorization
- Difficulty levels
- Step-by-step solutions
- Search and filtering

### 12. **Boot Recovery**
- Boot repair for Windows PCs
- Boot recovery for Macs
- Recovery media creation
- Boot troubleshooting workflows

---

## Technical Stack

- **Frontend**: React + TypeScript + Tauri
- **Backend**: FastAPI (Python)
- **Database**: SQLite (MVP), Postgres (Production ready)
- **Rust Services**: Modular services architecture
- **Design System**: Custom CSS with animations and utilities

---

## Key Principles

- **Compliance-First**: All operations are legal and auditable
- **Analysis Over Action**: Analyzes and routes, doesn't execute circumvention
- **Ownership Respecting**: Verifies ownership before operations
- **Fully Auditable**: Complete audit trail of all activities
- **Professional Grade**: Built for repair shop operations
