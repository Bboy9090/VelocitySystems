# App Store Review Notes (macOS / Desktop)

## App Purpose
Compliance-first device analysis and legal routing platform.
No device modification capabilities.

## Restricted APIs
None used.

## User Data
- No credentials collected
- No device content accessed
- Non-invasive metadata only
- Audit logs (action labels, timestamps)

## Functionality Clarification
The app does not unlock, bypass, or alter devices.
It generates documentation and guidance for lawful pathways.

## Screenshots Should Show
- Device Insight (analysis view)
- Compliance Summary (report view)
- PDF Export (compliance record)

## Screenshots Should Avoid
- Any language implying action
- "Unlock" or "Bypass" terminology
- Tool references
- Execution language

## Key Compliance Statements
- "Analysis and documentation only"
- "No modification or circumvention performed"
- "External authorization may be required"
- "All activity logged for compliance"
