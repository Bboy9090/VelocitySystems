# Tool-Agnostic Glossary (Public-Safe)

## Device Insight
A read-only summary of observed device metadata and protection posture.

## Ownership Confidence
A confidence score derived from documentation and attestations.
Confidence does not grant permission.

## Legal Context
Jurisdiction-aware classification describing whether recovery is permitted, conditional, or prohibited.

## Interpretive Review
A gated analytical view for complex cases that provides historical and legal context without steps or actions.

## External Authorization
Approval required from a manufacturer, carrier, executor, or court before any recovery attempt.

## Compliance Record
A formal PDF documenting analysis, rationale, and audit references.

## Security Research Landscape
Publicly discussed research activity in the security community.
Referenced for awareness and policy analysis only.

## Capability Class
Abstract category describing device protection characteristics.
Not a method or tool.
