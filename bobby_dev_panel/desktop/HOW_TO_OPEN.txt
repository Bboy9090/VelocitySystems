═══════════════════════════════════════════════════════════════

⚠️ FIRST TIME SETUP:

   Before running, you need:
   
   ✓ Node.js (download from https://nodejs.org/)
   ✓ Rust (install from https://rustup.rs/)
   ✓ Python 3.8+ (usually already installed)
   
   The launcher will check these for you!

═══════════════════════════════════════════════════════════════

💡 WHAT HAPPENS:

   1. Launcher checks prerequisites
   2. Installs dependencies (first time only)
   3. Compiles Rust backend (first time: 5-10 min)
   4. Opens app window (1400x900)
   5. Shows dashboard with device status

═══════════════════════════════════════════════════════════════

📦 TO CREATE INSTALLABLE APP:

   After running once, you can build:
   
   npm run tauri build
   
   Creates installer in:
   src-tauri/target/release/bundle/msi/
